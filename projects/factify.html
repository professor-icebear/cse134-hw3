About Factify.AI

Project Duration: January 2025 â€“ February 2025

Location: London, UK

Factify.AI is an AI-powered real-time fact checker that verifies image, text, or URL inputs and generates detailed reports with reliability scores, reasoning, corrections, false claims, and counter sources.

The application was also converted into a one-click Chrome extension for seamless fact-checking of current webpages.

Key Features

  • Multi-input support: verify images, text, or URLs
  • Real-time fact-checking with AI-powered analysis
  • Detailed reports including reliability scores and reasoning
  • Corrections and identification of false claims
  • Counter sources provided for verification
  • One-click Chrome extension for webpage fact-checking

Technologies Used

  • Frontend: TypeScript, Next.js, Tailwind CSS
  • Backend: Next.js API routes
  • Web Scraping: Cherio, Axios
  • Extension: Chrome Extension Tools
